# polynomial

### Add the dependency:
 - libarary for Scala @SCALA@ (JS, JVM, and Native platforms)
 - depends on cats @CATS@ (for the `Store` monomial)
 
```scala
"com.julianpeeters" %% "destructured" % "@VERSION@"
```
